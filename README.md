# Ahmed Portfolio

A modern, dark-themed single-page personal portfolio website built as a self-contained static site.

## Files

- `index.html` — page content and structure
- `styles.css` — responsive dark theme and layout

## Deploy for free

### GitHub Pages

1. Create a GitHub repository.
2. Upload `index.html`, `styles.css`, and this `README.md`.
3. Open repository settings.
4. Go to **Pages**.
5. Deploy from the `main` branch root.

### Netlify

1. Go to Netlify and choose **Add new site**.
2. Drag and drop this folder, or connect the GitHub repository.
3. No build command is required.
4. Publish directory: `/`.

## Customize

Update the name, email, project descriptions, and social links in `index.html`.
